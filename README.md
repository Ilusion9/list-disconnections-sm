# sm_last command
This plugin will display informations of disconnected players through sm_last command.

# Output example
```
Last disconnections
-------------------------
01. STEAM_1:1:123456XX "decoy" - 50s ago
02. STEAM_1:1:123456XY "Ilusion9" - 3m ago
03. STEAM_1:1:123456YZ "Fallen" - 1h 47m ago
```

# Cvars
```
sm_last_size 10
```
